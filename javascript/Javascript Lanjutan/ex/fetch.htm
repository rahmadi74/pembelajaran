FETCH adalah sebuah method pada API javacript untuk mengambil resources dari jaringan, dan mengembalikan promise yang selesai (fullfilled) ketika ada response yang tersedia.

 

Di bawah ini penggunaan dari fetch untuk menggantikan penggunaan ajax jQuery dari project Latihan Callback

1.       html

2.       javascript

 

fetch (resources, init);

RESOURCES:

1.       URL -> alamat dari sumber data yang kita ambil

2.       Request Object -> representasi permintaan sumber

 

INIT (optional) adalah konfigurasi tambahan pada sebuah request berbentuk object

·       method  (GET, POST…)

·       headers

·       body

·       mode

·       cache

·       referrer

·       referrerPolicy

·       integrity

·       keepalive

·       signal

 

RESPONSE adalah hasil dari fetch berupa promise, ada 2 yaitu Response Property dan Response Method

Response Property:

·       headers

·       ok

·       redirected

·       status

·       statusText

·       type

·       url

·       body

Response Method

·       clone()

·       error()

·       redirect()

·       blob()

·       formData()

·       json()

·       text()
