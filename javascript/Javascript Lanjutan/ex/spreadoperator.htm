SPREAD OPERATOR adalah operator yang memecah (expand/unpack) iterables menjadi single element

Iterable Object:

1.  string  2.  Array  3. Arguments  4. NodeList  5. Typed Array  6. Map 7. Set  8. User-defined Iterable

 

contoh:

const mhs = [‘Sandhika’, ‘Doddy’, ‘Erik’];

console.log (…mhs);         // Sandhika  Doddy  Erik

console.log (…mhs[0]);    // S a n d h i k a

 

PENGGUNAAN SPREAD OPERATOR ( … )

1.       Menggabungkan 2 Array atau lebih

const mhs = [‘Sandhika’, ‘Doddy’, ‘Erik’];

const dosen = [‘Ade’, ‘Hendra’, ‘Wanda’];

const orang = […mhs, …dosen];

atau bisa juga menggunakan concat

const orang = mhs.concat(dosen);

console.log (orang);

2.       Meng-Copy Array

const mhs = [‘Sandhika’, ‘Doddy’, ‘Erik’];

const mhs1 = […mhs];   //bedakan dengan const mhs1 = mhs; (yg dicopy referensinya)

3.       Mengubah NodeList menjadi Array

const liMhs = document.querySelectorAll(‘li’);

const mhs = […liMhs].map(m =>m.textContent);

console.log (mhs);

4.       Mengubah String, Arguments menjadi Array -> lihat no 3
