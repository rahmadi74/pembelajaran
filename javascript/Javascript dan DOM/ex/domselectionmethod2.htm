QUERY SELECTOR  DAN QUERY SELECTOR ALL

·       selectornya menggunakan selector yang sama dengan selector css

·       file html nya disini

 

1.       querySelector()

hasil: element

const p4 = document.querySelector('#b p');

p4.style.color = 'green';

p4.style.fontSize = '30px';

const li2 = document.querySelector('section#b ul li:nth-child(2)');

li2.style.backgroundColor = 'orange';

apabila querynya menghasilkan  banyak element, yang diambil hanya element pertama

const p = document.querySelector (‘p’);

p.innerHTML = ‘Ini diubah dari javascript’;

 

2.       querySelectorAll()

hasil: nodeList

const p = document.querySelectorAll('p');

for (let i=0; i<p.length; i++){

    p[i].style.backgroundColor='lightblue';

}

 

 

MENGUBAH NODE ROOT

const sectionB = document.getElementById('b');   //node root document

const p4 = sectionB.querySelector('p');                    //sekarang node root nya diganti sectionB

p4.style.backgroundColor = 'orange';
