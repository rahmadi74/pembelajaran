Ada 3 cara penulisan tipe data di go

1.       Deklarasi variabel beserta tipe data (manifest typing)

var lastName string

var firstName string = "john"

 

2.       Deklarasi variabel tanpa tipe data (type inference)

var firstName = "john" // menggunakan var, tanpa tipe data, menggunakan perantara "="

lastName := "wick" // tanpa var, tanpa tipe data, menggunakan perantara ":="

lastName = "ethan" // tanda “:=” hanya digunakan sekali selanjutnya cukup “=”

catatan:  tanda “:=” hanya bisa digunakan dalam blok fungsi

 

3.  Deklarasi Multi Variabel

var first, second, third string

first, second, third = "satu", "dua", "tiga"

atau lebih ringkas:

first, second, third := "satu", "dua", "tiga"

 

4.  Variabel Underscore (Untuk menampung nilai yang tidak digunakan / keranjang sampah)

5.  Deklarasi variabel menggunakan keyword “new”  (Untuk membuat pointer dengan tipe data tertentu)

name := new(string)

fmt.Println(name)   // 0x20818a220

fmt.Println(*name)  // ""

 

6.  Deklarasi variabel menggunakan keyword “make”

Hanya digunakan untuk pembuatan varibel-variabel berikut:

·       channel

·       slice

·       map
