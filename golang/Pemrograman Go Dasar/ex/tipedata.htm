Tipe data golang

1.       Tipe Data Numerik

·       Non Desimal

Tipe data            Cakupan bilangan

uint8     0 ↔ 255

uint16   0 ↔ 65535

uint32   0 ↔ 4294967295

uint64   0 ↔ 18446744073709551615

uint       sama dengan uint32 atau uint64 (tergantung nilai)

byte      sama dengan uint8

int8       -128 ↔ 127

int16     -32768 ↔ 32767

int32     -2147483648 ↔ 2147483647

int64     -9223372036854775808 ↔ 9223372036854775807

int          sama dengan int32 atau int64 (tergantung nilai)

rune      sama dengan int32

·       Desimal ( rujukan IEEE-754 32-bit floating-point numbers)

float32

var decimalNumber = 2.62  // decimalNumber adalah float32

fmt.Printf("bilangan desimal: %f\n", decimalNumber)      // template %f digunakan untuk data desimal menjadi string

fmt.Printf("bilangan desimal: %.3f\n", decimalNumber)  // %.3f akan menghasilkan 3 digit desimal

float64

 

2.       Tipe Data String

·       diapit tanda petik dua (Quote)

var message string = "Halo"

fmt.Printf("message: %s \n", message)

·       diapit tanda backticks

var message = `Nama saya "John Wick".

Salam kenal.

Mari belajar "Golang".`

fmt.Println(message)

 

3.       Tipe Data bool / boolean (nilai true atau false)

var exist bool = true

fmt.Printf("exist? %t \n", exist)   // templat %t digunakan untuk memformat tipe data bool menjadi string

 

4.       Nilai Zero Value dan nil

·       nilai Zero Value (nilai awal)

Zero value dari string adalah "" (string kosong).

Zero value dari bool adalah false.

Zero value dari tipe numerik non-desimal adalah 0.

Zero value dari tipe numerik desimal adalah 0.0.

·       nilai nil (nilai kosong)

tipe data yang bisa diisi nilai nil sbb:

pointer

tipe data fungsi

slice

map

channel

interface kosong atau interface{}
