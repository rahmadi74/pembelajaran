GRID AREA & GRID GAP

A.      GRID AREA

 

grid-template-area

Mendefinisikan grid template menggunakan nama dari area yang ditulis pada property grid-area pada item

.container{

       grid-template-areas:

              “ <nama-area> | none | ’.’ “

              “ <nama-area>  <nama-area> “;

}

contoh:    html     css

 

grid-template

shorthand untuk mendefinikan grid-template-rows, grid-template-colums dan grid-template-areas dalam satu deklarasi

grid-template: <grid-template-rows> | <grid-template-areas> / <grid-template-columns>;

contoh:    html    css

 

B.      GRID GAP

 

grid-column-gap & grid-row-grap                          -> cara lama

grid-gap: <grid-row-gap> <grid-column-gap>      -> shorthand lama

column-gap & row-gap                                            -> cara baru

gap: <grid-row-gap> <grid-column-gap>              -> shorthand baru

column-gap: 10px;

row-gap: 10px;

gap: 5px 5px;   atau   gap: 5px;

contoh:    html    css
