FLEXBOX

Model Layout 1 Dimensi yang dapat mengatur jarak dan penjajaran antar item dalam sebuah kontainer

 

1 Dimensi?

Hanya dapat mengatur 1 dimensi pada saat tertentu, antara baris atau kolom, tidak dapat keduanya sekaligus

 

contoh:

·       html

·       css

 

PROPERTY CSS FLEXBOX PADA CONTAINER:

1.       display

fungsi: membuat element parent menjadi flexbox dan element didalam berperilaku flex juga

display: flex;   /*atau inline-flex*/

2.       flex-direction

fungsi: Untuk mengatur arah items pada kontainer (default row)

flex-direction: row;

3.       flex-wrap

fungsi: Memungkinkan memindahkan Items ke baris di bawahnya

flex-wrap: wrap;

4.       justify-content

fungsi: Mengatur jarak antar items secara horisontal

justify-content: center;

5.       align-items -> akan berfungsi ketika height pada container ditentukan nilainya

fungsi: Untuk mengatur kesejajaran antar items secara vertikal

align-items: center;

6.       align-content -> akan berfungsi ketika height pada container ditentukan nilainya

fungsi: mengatur jarak antar items secara vertikal (berfungsi ketika items lebih dari 1 baris)

align-content: center;

 

PROPERTY CSS FLEXBOX PADA ITEMS:

1.       order

.satu {order: 3;}

2.       flex-grow / flex -> jika digunakan biasanya width & height items tidak diatur

fungsi: untuk mengatur ukuran items secara otomatis

.container div.dua{flex: 3;}

3.       align-self

fungsi: untuk mengatur 1 item saja untuk diposisikan kemana..

.container div.dua{flex: 3; align-self: flex-end;}
